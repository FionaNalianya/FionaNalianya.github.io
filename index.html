MINIATURE ROVER

The purpose of this project was to create a mobile multipurpose modular bot that can be adapted to various tasks easily. It was mainly used for surveillance in blindspots in a CCTV covered area.

In a CCTV monitored space, it is sometimes difficult to get 100% coverage due to blindspots caused by obstacles or space architecture. To combat this, we postulated that it may be necessary to create a mobile miniature rover that can be moved in the facilities to aid the security personnel get more coverage, especially at night. This was done using basic sensors, actuators and microcontrollers from the Arduino starter kit.

  • Client: Gearbox Kenya

Audio Processing With Python

The purpose of this project was to use python to process audio files

The audio processing was done by loading a wav file to produce audio signal parameters such as: the number of channels in the wav file, the sample width, the frame rate/sample rate, the number of frames and the values of a frame. I was then supposed to plot the wavfile object using matplotlib, capture an audio with my microphone and plot its wavfile object using a python library of my choice, transcriibe the audio file and encrypt and decrypt the input.

  • Client: Audio Processing
  • Category: Python

Lock Controller

To automate the process of locking and unlocking the Gearbox Academy cupboard store, we decided to create an automatic lock controller system that will be able to store a database of users that will access the cupboard and electronically access it.

The purpose of this project was to create an automatic lock controller system using MQTT messaging protocol. We settled with mosquitto broker since it supported TLS and the existence of plugins that could enable the integration of a databse, which was quite reliable as compared to other brokers such as RabbitMQ, EMQ and VerneMQ. To set up the MQTT, we installed mosquitto broker, installed mosquitto clients and created a config file in mosquitto andnamed it default. For the Arduino IDE, we installed pubsubclient libraries and ESP8266 board packges

  • Client: Gearbox Kenya
  • Category: IoT-based Design
