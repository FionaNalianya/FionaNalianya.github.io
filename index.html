Regression Analysis

The purpose of this project was to provide advice to home owners about how home renovations might increase the estimated value of their homes using multiple linear regression to predict the estimated value of homes in King County.

I created a baseline model, a feature selection model, a linear regression model and a polynomial regression model. I then compared the Root Mean Squared Error and variance of each model as compared to the baseline model to determine which model works best for this project.

  • Client: King County, California

Predictive Modeling of Accident Severity in Addis Ababa

The Addis Ababa Urban Planning Institute aims to enhance safety by identifyng high-risk zones through accident severity classifications, influencing road design and traffic flow management. Precise accident severity classification aids in assessing safety measures and formulating effective policies.

I collaborated with my colleagues and used Ensemble Methods for modeling and found that the Random Forest Classifier was the best model in terms of precision, recall, accuracy and f1-score. We then performed hyperparameter tuning and further feature selection to improve the accuracy of the model.

  • Client: Addis Ababa Urban Planning Institute

MovieLens Recommendation System

The purpose of this project is to develop an effective movie recommendation system that leverages collaborative filtering and content-based filtering techniques based on user ratings and movie attributes. The goal was to provide personalized movie recommendations to users, addressing the cold start problem for new users and enhancing the user experience by suggesting movies tailored to their preferences and viewing history.

To implement this, I performed collaborative filtering, incorporating both item-based and user based filtering with the Root Mean Squared Error as the main evaluation metric. I then performed collaborative filtering to address the cold start problem.

  • Client: MovieLens
